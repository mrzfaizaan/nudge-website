# Pricing — Nudge

## Product
Nudge is an always-on-top task HUD for Windows. It reads a single .md (Markdown) file and floats tasks above every window. No accounts, no cloud, no Electron. Built in Rust with Tauri v2.

## Features Included
- Always-on-top HUD banner
- Auto-hide to 6px strip on mouse-away
- Multi-monitor support
- Global keyboard shortcut (Ctrl+Alt+Shift+N)
- Light/dark theme
- Drag-drop task reordering
- Right-click context menu
- Inline task editing
- Auto-scroll with configurable cadence
- Hot reloading (detects external file edits)
- LLM formatting assistant (copy prompt to clipboard)
- 3 task types: Action (checkbox), Bullet (dot), Plain (text)
- Numbered list support
- Task counter and progress bar
- Quick file switcher (recent files)
- Settings: position, monitor, font size, width, opacity
